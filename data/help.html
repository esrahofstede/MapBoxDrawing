Center

In het menu staat de knop Center. Hij centreert dan op Deventer

Zoom in op een gebied

Hiervoor zijn twee mogelijkheden. Je kan in de menubalk via meta -> zoom naar gebied in zoomen door te zoeken op naam van een gebied. Dit is de naam + .kml erachter. Het alternatief vind ik makkelijker. Dit is Open -> File en dan selecteer je het gebied uit de Google Drive map. Hiervoor moet je Google Drive wel geinstalleerd hebben. Na het selecteren van het bestand zie je de contouren van het gebied. Door via meta -> clear te doen verdwijnt het gebied en kan je gaan tekenen. Het maximale zoomvniveau is 20. Mocht je meer nodig hebben, dan kan ik dat regelen.

Het tekenen gebeurt via de icoontjes aan de rechterkant van de kaart. Hiermee kan je een polygon tekenen door middel van de vijfhoek. De kleuren zijn tijdens het tekenen roze. Na het tekenen klik je op een van de randen en dan op save en de kleur is blauw en de binnenkant van de polygon doorzichtig. Het bloknummer kan je toevoegen door een marker te maken. Na het selecteren van de marker kan je het getal invullen als symbool. De grote is aan te passen door de size te veranderen naar large. Het polygon kan je wijzige via de edit knop. Die zit aan de rechterkant van de kaart. De kaart is ook op volledig scherm beschikbaar, door op het pijltje te klikken links van JSON (in de bovenrand). Hierdoor verdwijnt de GeoJSON.

Opslaan

Je kan de map opslaan via Save -> GeoJSON. De kaart kan je later ook weer openen door hem in te laden met File -> Open.

Protips?

  • cmd+s: save map to github gists
  • cmd+a: download map as geojson
  • arrow keys: navigate the map
